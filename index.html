Pantry App

Created a pantry management system for my second week project of the Headstarter SWE Fellowship. I utilized Next.js, Firebase and Firestore for authentication and data storage, Material UI for the styling of the app, and I deployed using Vercel. The app allows you to add multiple iterations of items with different expiration dates, search items in the pantry, and remove items. If an item is present with multiple expiration dates it will automatically remove the iteration of the item closest to expiration. I also implemented a function that uses OpenRouter AI API key to suggest recipes based on the items in your pantry. The recipe generation function of this project is not perfect, but I intend to perfect it using what I'm currently learning in the the fellowship.
